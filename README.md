# Experts CLI - A Golang Command Line Interface

#### Download the binary or compile yourself and run helper for more information
```bash
$ ./expertscli --help
```
<br/>

- <b>Entrypoint</b>
    - main.go or binary
- <b>Commands and flags</b>
    - Start
        - port: int
        - version: string*
    - Message
        - text: string*

<br/>
<h5>*not null</h5>
